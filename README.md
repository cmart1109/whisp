# Overview

This project is a simple social media clone, like x, where you can write some posts and comment posts from others. You can login to do it and like posts.

This software was written to communicate your ideas and share some fun jokes with others!

[Software Demo Video](http://youtube.link.goes.here)

# Web Pages

I create some pages here
- Home page to see all posts
- Login page 
- Register Page

# Development Environment

I used React to Make this project, using typescript and Nextjs Library, prisma for the database, and chakra-UI for the interface
